# Low Poly Mediaval Europe Weapon Pack - Documentation

Thank you for purchasing the Low Poly Mediaval Europe Weapon Pack!
This package contains 9 high-quality low poly weapons ready for your game.

---------------------------------------------------------------------
[ 1. Content ]
- Prefabs: 9 Characters (Sword, Axe, etc.)
- Models: FBX format
- Textures: Color textures included

[ 2. How to Use ]
1. Open the "Prefabs" folder in the Project window.
2. Drag and drop the character prefabs directly into your scene.
3. To animate the characters, please use any standard Unity Humanoid animations (Animator Controller).

[ 3. Technical Info ]
- Render Pipeline: Built-in (Standard Shader)
